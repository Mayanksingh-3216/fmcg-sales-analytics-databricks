
# FMCG Sales Analytics Pipeline

End-to-end data lakehouse on Databricks analyzing ₹104B revenue across 595M transactions.

## Features
- Medallion Architecture (Bronze → Silver → Gold)
- Star Schema with fact + dimension tables
- Interactive BI dashboard

## Tech Stack
Databricks • PySpark • Delta Lake • AWS S3 • Unity Catalog

## Notebooks
1. Setup & date dimension
2. Dimension processing (customers, products, pricing)
3. Fact processing (full load + incremental)

## Key Metrics
- ₹103.98B total revenue
- 101 products analyzed
- 100+ customers tracked
- 6 interactive visualizations
